# Parabellum Feed — Handoff 23/06/2026

## O que é o projeto
Feed RSS próprio hospedado no Render (`https://parabellum-feed.onrender.com`) que alimenta o Meta Commerce Manager com imagens 1:1 das estampas. Substitui o catálogo gerado pelo app oficial da Meta/Shopify que usava imagens de cabide e IDs incompatíveis.

## Stack
- `server.js` → Node/Express no Render → gera XML (Google Base namespace `g:`)
- Shopify (parabellum-br.myshopify.com) → fonte dos produtos e imagens
- Imagens 1:1 ficam no CDN da Shopify com `altText = "meta-catalog"`
- Meta Commerce Manager → "Parabellum Catalog" (ID: 1937394170168900) → consome o feed
